# Product Information Quality Dashboard

A data app built with Python and Streamlit that analyzes 
how complete and clear product information is, and flags 
items that might confuse customers.

## 🚀 Live Demo
https://example-info-quality.streamlit.app/

## 🛠️ Built With
- Python
- Streamlit
- SQLAlchemy & SQLite
- Pandas

## ✨ Features
- Add products with category-specific attributes
- Automatically scores product information quality
- Flags products with incomplete information
- Export product data to CSV
- Filter products by category

## 🏃 Run Locally
```bash
git clone https://github.com/Oaazu/product-info-quality
cd product-info-quality
pip install -r requirements.txt
streamlit run app.py
```
